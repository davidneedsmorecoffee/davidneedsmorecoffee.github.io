Summary: (Y) vs. Latitude (X)

This webpage provides an overview of results from the LATITUDE project. One main objective of this project is to examine how specific weather parameters, e.g., temperature, varies with latitude.

In this project, a list of coordinates was first randomly generated based on pre-specified longitude and latitude ranges. Based on the longitudes and latitudes, nearest cities were identified. An API call was performed (to the openweathermap.org website) to obtain additional information for each city, including country, maximum temperature (Fahrenheit; F), humidity (%), cloudiness (%), and wind speed (miles per hour; mph).

The data were pulled, processed, analyzed, and visualized in JupyterLab via Python, using libraries such as matplotlib, pandas, and numpy. Please use the navigation bar (see right top corner) to see i) Plots: more details of each analysis, ii) Comparisons: overview of the four main analyses, and iii) Data: over 500 cities worth of weather-related data used in the project. The visualization box also provides quick access to each analysis.
